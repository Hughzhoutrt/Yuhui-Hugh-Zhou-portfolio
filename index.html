Some Things I've Made

Pokemon-Guess-Em-All

My first React application. Pokemon guess and catch Application. Multiple APIs and Firebase applied. Users are shown random Pokemon shadows and are prompted to guess their names. They also can check the Pokédex, catch the Pokémon and manage their Pokémon teams. Using Firebase, the Pokémon teams are updated in real-time based on the users' verified usernames.

Live Repo

What-to-Watch

Movie Watch List Application. Multiple APIs applied. Users can request randomly generated movies then save and store them through a drag and drop feature. Buttons allow users to add or remove films from their watchlists if they change their minds. Movie details are available in pop-up windows when movie cards are clicked.

Live Repo

YUHUI-WEATHER-NETWORK

Weather checking and GIF generating Application. Multiple APIs applied. The user's city is automatically acquired, and the weather shows the application; the weather for other cities can be entered and checked. Checked weather data can be saved and marked on the page. Weather-relevant GIFs are generated to make users happy.

Live Repo

YUHUI-S-BREAK-TIMER

My first JavaScript application. Countdown Application. Multiple default countdown options. Theme colours and languages are selective. Both normal countdown and be-back countdown are applied. (Be-back function: entering the time you want to be alerted, and this application can calculate the countdown time. Maximum 24 hours in advance.

Live Repo
